# v0.1.0

Initial public-ready release of `visio-figure-redrawer`.

## Added

- Reference-first workflow for redrawing arbitrary figures as editable Visio diagrams.
- Project-aware scanner for paper/model directories.
- YAML-to-figure first-pass drafting for YOLO/RT-DETR/Ultralytics-like configs.
- Journal style profiles: reference-first, Nature-like, IEEE-like, Elsevier-like, IOP/SPIE-like.
- JSON figure spec schema, audit script, preview renderer, and Visio renderer.
- Environment check for Python dependencies and Microsoft Visio COM.

## Known Limitations

- Arbitrary Python `forward()` graphs require Codex interpretation.
- YAML drafting produces a first-pass architecture figure; polishing and reference-style matching still require iteration.
- Final journal compliance requires checking the target journal's current author/artwork guidelines.
- `.vsdx` rendering requires Windows and desktop Microsoft Visio.
